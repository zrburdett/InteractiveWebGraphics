Notes

I had to learn several concepts throughout this project to make sure that my illustrations and interactive graphics turned out the way I had hoped for.


To start, I mostly used CSS shapes and edited them using basic Javascript. However, I used a lot of jQuery to add, remove, and change classes and their attributes.


I also used SVG files for the Rook page. This is because I needed complex vector graphics to be displayed at a high resolution. It also made it very easy to change the colors as I only needed to change the fill color when the click event or mouse moved event was triggered.


To go above and beyond, I brought in my knowledge from my IDAPS class and made my CSS graphics interactive with if statements, conditionals, and various variables. I also used the framework, Materialize, to make my website easily mobile friendly and to give it a very nice and cohesive look.
